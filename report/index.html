RNA Secondary Sequence Prediction Algorithm

Problem Statement

Problem :

Given an RNA sequence s, predict it’s optimal secondary structure.

Formulation :
  • An RNA sequence s is a string of bases; s = b1 b2…..bn where bi ∈ {A,U,C,G} for 1 ≤ i ≤ n
  • The optimal secondary structure predicted for s contains a set of base pairings B = { (i,j) } where 1 ≤ i < j ≤ n and the pairings satisfy the following rules:
    • [valid pairing] (S[i],S[j]) must be either one of (A,U) or (U,A) or (C,G) or (G,C)
    • [matching] Each i can pair with at most one j.
    • [at least 4 intervening bases in-between a pairing] 4 < j-i < n
    • [no crossing or no-knot] For every (i,j) and (k,l) ∈ B where i < k we cannot have i < k < j < l
  • We have to find the maximum number of base pairings satisfying the above constraints for the given RNA sequence s.

Dynamic Programming solution

Recurrence relation:

Let OPT(i,j) denote the maximum number of pairings possible between the indices i and j. This value will be the maximum of { OPT(i,j-1) (when s[i] does not pair with s[j]) and the maximum number of pairs obtained considering a base at index t (i ≤ t ≤ j-5) pairs with a base at index j (= 1 + OPT(i,t-1) + OPT(t+1,j)) } Also, because there must be atleast 4 intervening bases between a pairing OPT(i,j) = 0 whenever j-i < 5.

We have to find the optimal secondary structure for the entire sequence, thus the maximum number of pairings is OPT(1,n) which can be obtained by solving the above recurrence.

To obtain the matching pairs we recursively trace back the corresponding (i,j) for which the max value was obtained while calculating the OPT(i,j) for the subproblems.
